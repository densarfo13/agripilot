# Farroway — Digital Verification for Agricultural Credit

---

## What It Does

Farroway is a verification and decision platform for institutions that extend credit to smallholder farmers. It digitizes the entire application workflow — from farmer registration and GPS capture to fraud detection and portfolio reporting.

---

## Who It Is For

- Agricultural lenders and microfinance institutions
- NGOs running input credit or subsidy programs
- Farmer cooperatives managing internal lending
- Development programs funding smallholder agriculture

---

## How It Works

**1. Field Officer registers a farmer**
Captures contact details, farm location, crop type, and experience. Works on Android phones in the field.

**2. Application is submitted with evidence**
GPS coordinates captured with device accuracy. Farm boundary mapped by walking the perimeter. Photos uploaded — each one hashed for fraud detection.

**3. Automated verification scoring**
Each application is scored 0-100 across five factors: GPS quality, boundary accuracy, evidence completeness, farmer profile, and application consistency.

**4. Fraud detection flags risks**
The system detects duplicate photos across applications, shared devices between farmers, and overlapping GPS coordinates. Flagged applications are escalated automatically.

**5. Reviewer makes a decision**
Reviewers see verification scores, fraud flags, and evidence in one view. They add notes, request field visits, and approve or reject with full documentation.

**6. Audit trail records everything**
Every action — creation, submission, GPS capture, review, decision — is logged with timestamps and user identity.

## Key Capabilities

| Capability | Description |
|-----------|-------------|
| GPS + Boundary Mapping | Device-level GPS capture and farm boundary measurement |
| Photo Hash Fraud Detection | Detects reused photos across different applications |
| Device Fingerprinting | Flags applications submitted from the same device |
| Verification Scoring | Automated 0-100 score with factor breakdown |
| Role-Based Access | Admin, reviewer, field officer, and investor views |
| Multi-Country Support | Supports multiple countries, currencies, and languages |
| Audit Trail | Complete log of every action for compliance |
| Portfolio Dashboard | Real-time analytics on decisions, risk, and disbursement |
| Android App | Works on low-cost Android phones in the field |
| Post-Harvest Tracking | Storage monitoring and market guidance for farmers |

---

## What Makes It Different

- **Fraud detection is built in**, not bolted on. Photo hashing and device tracking happen automatically on every application.
- **Verification is systematic**, not subjective. Every application gets the same scoring methodology.
- **The audit trail is complete**. Every action is logged. No gaps in accountability.
- **It works offline-capable on Android**. Field officers do not need constant connectivity.
- **It is institution-focused**. Built for the workflows NGOs, lenders, and cooperatives actually use.
